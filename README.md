# 🤖 Automated Lead Generation Report

Automatically scrapes Google Maps, cleans data and generates
a professional Excel report — all in one single command.

## 💼 What Problem This Solves
Agencies spend days manually collecting and cleaning leads.
This tool does everything automatically in under 30 minutes.

## ⚙️ What It Does Step By Step
- Step 1 → Scrapes businesses from Google Maps
- Step 2 → Cleans and scores lead quality (Hot/Good/Cold)
- Step 3 → Generates professional Excel report with 3 sheets
